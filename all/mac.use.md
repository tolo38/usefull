Create a shortcut
---

  - System Preferences > Keyboard Settings > Modifier Keys 

- With automator
  - choose service
  - find Launch Application
  - select the app
  - set "Service receives selected" to "no input"
  - save
  - then open parameter->keyboard->shortcut->service
  - set a shortcut to the new services
